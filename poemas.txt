**El color del server** 
Las rosas son rojas 
las violetas azules 
y este server esta muerto

**Preguntas** 
Cada vez que pienso en ti 
mis ojos rompen en llanto 
y muy triste me pregunto 
¿por qué te este server esta muerto?

**Entre lagrimas** 
Llorando tomé el lápiz 
llorando te escribí 
Llorando te suplico 
que no dejes a este server morir

**Soledad Virtual** 
Me dicen 
tienes al server por delante 
pero yo miro 
y no veo a nadie

**Elegía digital** 
Silencio en la sala, 
ni un saludo dan ya. 
El server ha caído, 
y nadie sabe por qué se va.

**Lamento del admin**
Horas de juego, 
y mensajes por red, 
pero el server murió… 
y yo sin vida otra vez.

**Apagón** 
En la noche oscura 
general dejó de brillar. 
Murió el server callado, 
sin siquiera avisar.

**Breve obituario** 
Aquí yacía un server fiel, 
con miembros de noble estirpe. 
Cayó por culpa de un admin cruel, 
pero el recuerdo aun persiste.

**Eco digital** 
Dije “hola” anoche, 
nadie contestó. 
Hoy borré el mensaje… 
y nadie lo notó

**En linea** 
Cinco conectados, 
pero nadie habla. 
Como fantasmas verdes 
rondando la sala.

**Bot fiel** 
Todos se fueron ya,
pero el bot quedó. 
Saluda con su script 
a quien nunca llegó.
